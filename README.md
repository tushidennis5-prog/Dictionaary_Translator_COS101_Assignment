# English-French Dictionary App

A lightweight Streamlit web application that serves as a quick-reference English to French dictionary. This tool allows users to input common English words and receive their French translations instantly.

## Features
* **Instant Search:** Get translations immediately upon clicking search.
* **Input Sanitization:** Handles case sensitivity and trailing spaces automatically.
* **Error Handling:** Provides clear feedback if a word is missing from the library.
* **Simple UI:** Minimalist interface built for speed and ease of use.
